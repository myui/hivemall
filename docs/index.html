Hivemall

Hivemall is a scalable machine learning library that runs on Apache Hive. Hivemall is designed to be scalable to the number of training instances as well as the number of training features.

Vision

Machine Learning made easy for SQL developers (ML for the rest of us) Interactive and Stable APIs w/ SQL abstraction This SQL query automatically runs in parallel on Hadoop

Features

Hivemall provides various machine learning functionality as well as feature engineering functions through UDFs/UDAFs/UDTFs such as classifier, regressor, anomaly detector and implementation for recommendations.
